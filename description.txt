                         МОДУЛЬ ТУ
Соглашение о названиях релле управления, выполнения ТУ и входов проверки
- Релле управления TY - ty_rel;
- Релле управления проверки наличия напряжения оперативного тока - v_rel;
- Груповое релле (релле которое комутирует Релле управления TY к выводам) gryp_rel; 
- Вход проверки наличия напряжения оперативного тока - v_input;

ПРОВЕРКИ И КОРОТКИЕ АЛГОРИТМЫ

1)              проверка наличия напряжения оперативного тока:
- подключить релле v_rel
- выполнить чтение v_input (ожидаеться 1)
- отключить релле v_rel

2)              проверка отсутствия залипания контактов релле:
- отключить релле v_rel
- выполнить чтение v_input (ожидаеться 0)

3)             проверка включения релле управления TY:
- отключить релле v_rel
- включить релле релле управления TY
- выполнить чтение v_input (ожидаеться 1)

4)             проверка включения групового релле (после выполнения проверки 3):
- отключить релле v_rel
- включить груповое релле
- выполнить чтение v_input (ожидаеться 0)

5)             проверка отключения групового релле (после выполнения проверки 4):
- отключить релле v_rel
- отключить груповое релле
- выполнить чтение v_input (ожидаеться 1)

6)             проверка отключения релле управления TY:
- отключить релле v_rel
- отлючить релле релле управления TY
- выполнить чтение v_input (ожидаеться 0)


Пункты 1-2 выполняються цыклически непрерывно
Пункты 3-6 выполняються в процесе срабатывания ТУ типа DP
Во время выполнения ТУ типа SP контроль НЕ ВЫПОЛНЯЕТЬСЯ
